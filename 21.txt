#### Понятие файл-сервера
Файл-сервер — это компьютер или специализированное устройство, предназначенное для хранения файлов и предоставления к ним доступа другим компьютерам в сети. Он обеспечивает централизованное хранение данных и управление доступом к файлам.

#### Отличие региональной сети от локальной
1. **Локальная сеть (LAN)**:
   - Ограниченная географически область (например, одно здание или небольшая территория).
   - Обеспечивает высокую скорость передачи данных.
   - Примеры: домашняя сеть, офисная сеть.

2. **Региональная сеть (MAN)**:
   - Охватывает большую территорию, как правило, город или регион.
   - Используется для соединения нескольких локальных сетей.
   - Скорость передачи данных может быть ниже, чем в локальной сети.
   - Примеры: сети провайдеров, сети между офисами компании в разных городах.

### Практическое задание

Создадим базу данных «Турагентство» с тремя сущностями: тарифы, клиенты (таблица создается при помощи SQL) и туроператоры. Затем напишем SQL-запрос, который выводит клиентов, отправившихся в путешествие в Феврале и отсортированы по фамилии.

```sql
-- Создание базы данных (в SQLite это можно пропустить, так как база данных создается при подключении)

-- Создание таблицы тарифов
CREATE TABLE Tarify (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    data_pol_eta TEXT,
    tsena_tura REAL
);

-- Создание таблицы клиентов
CREATE TABLE Klienty (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    familiya TEXT,
    imya TEXT,
    data_otpravki TEXT, -- Дата отправления
    id_tarifa INTEGER,
    FOREIGN KEY (id_tarifa) REFERENCES Tarify(id)
);

-- Создание таблицы туроператоров
CREATE TABLE Turoperatory (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    nazvanie TEXT
);

-- Запрос на выборку клиентов, отправившихся в путешествие в Феврале и отсортированных по фамилии
SELECT familiya, imya, data_otpravki
FROM Klienty
WHERE data_otpravki LIKE 'Фев% %' -- Используем LIKE для поиска вхождения "Фев" в строке даты
ORDER BY familiya;
```

Этот SQL-запрос выбирает данные из таблицы клиентов, чьи даты отправления начинаются с "Фев" (Февраль) и сортирует результаты по фамилии клиентов (`ORDER BY familiya`).
