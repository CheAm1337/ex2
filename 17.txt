### Вопросы

#### Понятие доступности
Доступность — это способность информационной системы обеспечивать своевременный и непрерывный доступ к информации и ресурсам для авторизованных пользователей. Высокий уровень доступности означает, что система минимизирует время простоя и максимально доступна для использования.

#### Какой оператор позволяет создавать диапазон значений в запросе
Оператор `BETWEEN` позволяет создавать диапазон значений в SQL-запросе. Он используется для выбора значений, которые попадают в указанный диапазон, включая границы диапазона.

#### Понятие информации
Информация — это данные, обработанные и представленные в контексте, который делает их значимыми и полезными для пользователя. Она может быть использована для принятия решений, обучения, анализа и других целей.

### Практическое задание

Создадим базу данных «Полицейский участок» с тремя сущностями: нарушители, сотрудники и отделения. Затем напишем SQL-запрос, который выводит нарушителей, которые проживают в городе Новосибирск и старше 18 лет.

```sql
-- Создание базы данных (в SQLite это можно пропустить, так как база данных создается при подключении)

-- Создание таблицы нарушителей
CREATE TABLE Narushiteli (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    familiya TEXT,
    imya TEXT,
    otchestvo TEXT,
    vozrast INTEGER,
    gorod TEXT,
    nakazanie TEXT
);

-- Создание таблицы сотрудников
CREATE TABLE Sotrudniki (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    familiya TEXT,
    imya TEXT,
    otchestvo TEXT,
    dolzhnost TEXT,
    otdelenie_id INTEGER,
    FOREIGN KEY (otdelenie_id) REFERENCES Otdeleniya(id)
);

-- Создание таблицы отделений
CREATE TABLE Otdeleniya (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    nazvanie TEXT,
    adres TEXT
);

-- Запрос на выборку нарушителей, которые проживают в городе Новосибирск и старше 18 лет
SELECT *
FROM Narushiteli
WHERE gorod = 'Новосибирск' AND vozrast > 18;
```

Этот SQL-запрос выбирает данные из таблицы нарушителей, которые проживают в городе Новосибирск и старше 18 лет (используя условия `WHERE gorod = 'Новосибирск' AND vozrast > 18`).
